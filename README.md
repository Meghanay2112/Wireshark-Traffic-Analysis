# Wireshark-Traffic-Analysis
# Task 5 – Capture and Analyze Network Traffic Using Wireshark

## Objective
To capture live network traffic and analyze different protocols.

## Tool Used
Wireshark

## Procedure
1. Installed Wireshark and Npcap.
2. Started capture on Wi-Fi interface.
3. Generated traffic by browsing websites and using ping command.
4. Stopped capture after 1 minute.
5. Applied filters like dns, tcp, udp, http.
6. Observed packet details and protocols.
7. Exported capture as .pcap file.

## Protocols Identified
1. DNS – Used for domain name resolution.
2. TCP – Reliable connection-oriented protocol.
3. UDP – Connectionless protocol used by DNS.
4. TLS/HTTP – Web communication protocol.

## Outcome
Successfully captured and analyzed live network packets.
